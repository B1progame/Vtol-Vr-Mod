# Security Policy

## Reporting a Vulnerability

Please do not open a public issue for security-sensitive reports.

Instead, contact the maintainer privately through the GitHub profile listed on the repository owner account, or open a minimal issue asking for a private contact method without posting exploit details.

## Scope

Security reports are most useful when they involve:

- Unsafe file operations.
- Unexpected deletion or modification of user data.
- Installer behavior.
- Auto-update behavior.
- Path traversal or profile import issues.

This project is a local desktop helper for installed VTOL VR Mod Loader Workshop content. It should never require secrets, Steam credentials, or game account credentials.
