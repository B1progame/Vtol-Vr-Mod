# Support

Use GitHub issues for bug reports, feature requests, and troubleshooting:

<https://github.com/B1progame/Vtol-Vr-Mod-Profiler/issues>

## Before Opening an Issue

- Make sure VTOL VR is closed before applying profiles.
- Make sure VTOL VR Mod Loader is installed from Steam.
- Make sure the mods referenced by a profile are installed locally.
- Try `Rescan` in the app after Steam finishes downloading Workshop content.
- Check the logs listed in the README.

## Useful Details to Include

- App version.
- Windows version.
- Whether you installed from a release build or built from source.
- Steps to reproduce the issue.
- Relevant log lines.
- Screenshots, if the problem is visual.
